# HackChat轻量版  
欢迎使用 HackChat轻量版，这是一个由[HackChat](https://github.com/hack-chat/main)改编的轻量级聊天室，适用于个人站长，用于临时开设聊天室服务器。  

## 如何使用  
### 先决条件  
- Node.js环境  
- 一个正常的网络环境  

### 开始使用  
1. 在本仓库目录下执行 `npm i` 来安装依赖  
2. 稍等片刻，按照配置向导来配置服务器  
3. 执行 `npm start` 来启动服务器，按 `Ctrl + C` 关闭服务器
